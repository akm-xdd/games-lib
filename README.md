# games-lib
 
This is a basic Vue.js and Flask App.

To run the backend server, use the following commands in order

```
cd backend
pipenv shell
pip install -r Requirements.txt
pip install uuid
python main.py
```

To run the frontend of the app, use the following commands in order

```
cd frontend
npm i
cd frontend
npm i
npm run serve
```

And that's it! 
Enjoy!
